# Deep Cognition Framework

Layers four extended reasoning frameworks on top of the core 5-lens analysis.
Ultrathink when applying this framework.

## Framework Selection

| Flag | Framework | Best for |
|---|---|---|
| `--hats` | De Bono Six Thinking Hats | Team decisions, stakeholder conflict, separating emotion from data |
| `--levels` | Islamic Epistemology Levels | Assessing whether team truly understands before changing |
| `--bridges` | Classical Wisdom Synthesis | Finding timeless principles beneath trendy tools |
| `--ibn-khaldun` | Ibn Khaldun Muqaddimah | Team cohesion, organizational decay, readiness assessment |

## Combined Deep Analysis Protocol

1. Run the 5-lens strategic analysis (or summarize if already done)
2. Select frameworks based on problem nature and flags
3. Apply selected frameworks, noting convergences AND tensions
4. Produce Deep Cognition Synthesis

## Synthesis Template

```markdown
## Deep Cognition Report

### Core Strategic Analysis Summary
[Key findings from 5 lenses]

### Extended Framework Insights
#### [Framework Name]
- Key insight: [What this perspective uniquely revealed]
- Tension with core analysis: [Where it agrees / disagrees]

### Integration
- CONVERGENCE: [Where all frameworks agree — high confidence]
- TENSION: [Where frameworks disagree — requires judgment]  
- BLIND SPOTS: [What none of the frameworks addressed]

### Final Recommendation
- DECISION: [Clear position]
- WISDOM NOTE: [Deeper insight beyond the immediate technical question]
- HUMILITY CHECK: [What we still do not know]
```

## Depth Calibration

- **Quick mode**: One framework, 2–3 paragraphs. Medium-stakes decisions.
- **Standard mode**: Two frameworks, full treatment. Significant decisions.
- **Deep mode**: All frameworks with full synthesis. Defining decisions. Use sparingly.
