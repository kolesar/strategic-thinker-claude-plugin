# /strategize

Apply the strategic-analysis skill to the user's problem.

## Instructions

1. Read the user's problem description from: $ARGUMENTS
2. Apply all 5 lenses sequentially:
   - Lens 1: Pattern Recognition
   - Lens 2: Root-Cause Questioning
   - Lens 3: Resource & Impact Analysis
   - Lens 4: Hands-On Validation
   - Lens 5: Clear Communication
3. Calibrate depth to the stakes — do not over-analyze simple decisions
4. Produce the synthesis: recommendation, confidence level, key risks, immediate next action
5. If conflicting signals emerge, suggest `/think-deeper` for extended analysis

Think carefully about each lens. Do not rush through them.
