# /think-deeper

Apply the deep-cognition skill with extended reasoning frameworks.

## Instructions

1. Parse the problem and optional flags from: $ARGUMENTS
2. Determine which frameworks to apply:
   - `--hats`: De Bono Six Thinking Hats only
   - `--levels`: Islamic Epistemology Cognitive Levels only
   - `--bridges`: Bridges Synthesis only
   - `--all`: All three frameworks with full synthesis
   - No flags: Auto-select based on problem nature
3. First run the 5-pillar strategic analysis as foundation
4. Layer selected extended framework(s) on top
5. Produce the Deep Cognition Synthesis
6. Include a WISDOM NOTE — the deeper insight beyond the immediate question

Ultrathink when applying this command.
