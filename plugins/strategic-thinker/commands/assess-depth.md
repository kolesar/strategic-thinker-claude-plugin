# /assess-depth

Evaluate understanding depth using the Islamic epistemology cognitive levels.

## Instructions

1. Read the system or problem description from: $ARGUMENTS
2. For each of the 7 levels (Hissī through Dhawqī), assess:
   - Do we have evidence of understanding at this level?
   - What specific artifacts or experiences demonstrate this level?
   - Where does our understanding stop?
3. Identify the current level honestly
4. Compare against the decision mapping — is the current level sufficient?
5. If there is a gap, recommend specific actions to deepen understanding

## Output Format

```
SYSTEM: [Name]
CURRENT LEVEL: [Number and name]
EVIDENCE: [What demonstrates this level]
DECISION BEING CONSIDERED: [What we want to do]
REQUIRED LEVEL: [Minimum for this decision type]
GAP: [What needs to happen before deciding]
```
