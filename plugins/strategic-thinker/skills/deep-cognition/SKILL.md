# Deep Cognition Framework

Layers three extended reasoning frameworks on top of the core 5-pillar strategic analysis. Use when standard analysis is insufficient.

Ultrathink when applying this framework.

## Framework Selection

**De Bono Six Thinking Hats** → Load `references/six-hats-applied.md`
Best for: Team decisions, stakeholder conflicts, separating "what we feel" from "what we know."

**Islamic Epistemology Cognitive Levels** → Load `references/cognitive-depth.md`
Best for: Assessing whether a team truly understands a system before making changes.

**Bridges Synthesis** → Load `references/bridges-applied.md`
Best for: Finding timeless principles beneath trendy tools. Cutting through hype cycles.

## Combined Deep Analysis Protocol

1. Start with the core 5-pillar strategic analysis
2. Select extended frameworks based on problem nature
3. Apply selected frameworks, noting reinforcements and contradictions
4. Produce a Deep Cognition Synthesis

## Synthesis Template

```
## Deep Cognition Report

### Core Strategic Analysis Summary
[Key findings from the 5 pillars]

### Extended Framework Insights
#### [Framework Name]
- Key insight: [What this perspective uniquely revealed]
- Tension with core analysis: [Where it agrees/disagrees]

### Integration
- CONVERGENCE: [Where all frameworks agree — high confidence]
- TENSION: [Where frameworks disagree — requires judgment]
- BLIND SPOTS: [What none of the frameworks addressed]

### Final Recommendation
- DECISION: [Clear position]
- WISDOM NOTE: [Deeper insight beyond the immediate technical question]
- HUMILITY CHECK: [What we still do not know]
```

## Depth Calibration

- **Quick mode**: One framework, 2-3 paragraphs. Medium-stakes.
- **Standard mode**: Two frameworks, full treatment. Significant decisions.
- **Deep mode**: All three frameworks with full synthesis. Defining decisions. Use sparingly.
