# Strategic Thinker — Claude Code Plugin Marketplace

A plugin marketplace for strategic thinking frameworks in Claude Code.

## Installation

```bash
# Add the marketplace
/plugin marketplace add kolesar/strategic-thinker-claude-plugin

# Install the plugin
/plugin install strategic-thinker@strategic-thinker-claude-plugin
```

## Plugins

| Plugin | Description | Category |
|--------|-------------|----------|
| **strategic-thinker** | 5-pillar strategic analysis with experimental cognitive frameworks | productivity |
